# Usage

<!-- usage -->

```sh-session
$ npm install -g terrastack-cli
$ terrastack COMMAND
running command...
$ terrastack (-v|--version|version)
terrastack-cli/0.0.1 darwin-x64 node-v10.10.0
$ terrastack --help [COMMAND]
USAGE
  $ terrastack COMMAND
...
```
